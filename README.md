## 🚀 Featured Projects

### 🔑 CyberKey (Password Scrambler)
A deterministic, hash-based password generator that eliminates the need for vulnerable databases. 
*   **Status:** [Live on Vercel](https://cyberkey.vercel.app)
*   **Security:** Built using PBKDF2/SHA-256 logic to "scramble" master keys into site-specific passwords.
*   **Upcoming:** Firefox Extension port for seamless browser integration.
